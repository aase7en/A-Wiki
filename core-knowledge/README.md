# Core Knowledge — Safeguarded Content Archive

> All original informational markdown files, tech notes, and wiki content from A-Wiki
> have been moved here to preserve the original structure and hierarchy.

## Structure

```
core-knowledge/
├── wiki/         ← Knowledge pages (concepts, entities, synthesis, sources)
├── docs/         ← Protocols, architecture, getting-started guides
├── decisions/    ← Architecture Decision Records (ADRs)
├── journal/      ← Daily journals
├── raw/          ← Raw source documents, data files
└── exports/      ← NotebookLM bundles
```

## Access

This directory is a **read-only archive**. The active operational layer is now at:

- `agent-skills/` — AI agent skills with Iron Law enforcement
- `skills/` — Legacy skills (maintained for backward compatibility)
- `scripts/` — Utility scripts
