# แนวทางการดูแลรักษาผู้สัมผัสโรคพิษสุนัขบ้า — สถานเสาวภา 2565

**ประเภท**: clinical guideline — สอดคล้อง WHO position 2018
**วันที่**: พ.ศ. 2565 (มิ.ย.)
**ผู้เขียน**: สถานเสาวภา สภากาชาดไทย (Queen Saovabha Memorial Institute)
**ขนาดไฟล์**: 1.6 MB

## ประเด็นหลัก

1. **TRC-ID regimen 2-2-2-0-2**: ฉีด ID 2 จุด ที่ D0, D3, D7, D28 (ไม่ฉีดวันที่ 14)
2. **RIG infiltration**: ฉีดรอบแผลทุกแผล — **ถ้าเหลือไม่ต้องฉีด IM ส่วนที่เหลือ** (เปลี่ยนจากแนวทางเก่า)
3. **Category II** (ช้ำ/ถลอกไม่มีเลือด): ให้เฉพาะวัคซีน ไม่ต้อง RIG
4. **กลุ่มพิเศษ**: HIV CD4<200 → ให้ระดับ Category III เสมอ แม้แผลเล็ก

## ข้อมูลที่น่าสนใจ / ขัดแย้งกับ CPG ของ DDC

- DDC ฉบับ 2564 ยังคงระบุสูตร 5 เข็ม IM (D0, D3, D7, D14, D28) เป็น default
- สถานเสาวภาแนะนำ TRC-ID 4 ครั้ง (D0, D3, D7, D28) — ตาม WHO 2018 dose-sparing
- ทั้งสองสูตรใช้ได้ในไทย แต่ TRC-ID ประหยัดวัคซีนมากกว่า
