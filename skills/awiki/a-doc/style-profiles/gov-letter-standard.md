## Layout: หนังสือภายนอก (External letter)

```
[ตราครุฑ — center]

ที่ {{หน่วยงาน}} {{เลขลำดับ}}                       {{ที่อยู่ผู้ส่ง}}
                                                  {{วันที่ เดือน พ.ศ.}}

เรื่อง   {{หัวเรื่องสั้น เป็นวลีคำนาม ไม่มีกริยา}}

เรียน   {{ตำแหน่งผู้รับ}}

อ้างถึง  {{ถ้ามี}}

สิ่งที่ส่งมาด้วย   {{ถ้ามี}}

         {{ย่อหน้า 1 — ขึ้นต้นด้วย "ด้วย..." หรือ "ตามที่..."}}

         {{ย่อหน้า 2 — รายละเอียด}}

         {{ย่อหน้าปิด — "จึงเรียนมาเพื่อโปรด..."}}

                                            ขอแสดงความนับถือ

                                            (ลายเซ็น)
                                            ({{ชื่อ-นามสกุล พร้อมคำนำหน้า}})
                                            {{ตำแหน่ง}}
                                            {{หน่วยงาน}}

{{หน่วยที่รับผิดชอบ / เบอร์โทร — มุมล่างซ้าย}}
```

## ลำดับชั้นคำขึ้นต้น (Salutation hierarchy — สำคัญมาก)

| ผู้รับ | คำขึ้นต้น | คำลงท้าย |
|---|---|---|
| พระมหากษัตริย์ | กราบบังคมทูล (ผ่านราชเลขา) | ด้วยเกล้าด้วยกระหม่อม ขอเดชะ |
| พระบรมวงศ์ฯ | ขอพระราชทานกราบบังคมทูล | ควรมิควรแล้วแต่จะโปรดเกล้าฯ |
| สมเด็จพระสังฆราช | กราบทูล | ควรมิควรแล้วแต่จะโปรด |
| พระสงฆ์ | นมัสการ | ขอนมัสการด้วยความเคารพอย่างยิ่ง |
| นรม./ประธานรัฐสภา/ประธานศาลฎีกา | กราบเรียน | ขอแสดงความนับถืออย่างยิ่ง |
| รัฐมนตรี/ปลัดกระทรวง/อธิบดี/ผวจ./นายอำเภอ/ผอ./หัวหน้า | เรียน | ขอแสดงความนับถือ |
| บุคคลทั่วไป | เรียน | ขอแสดงความนับถือ |

> ⚠️ **เลือกผิดระดับ → หนังสือถูกส่งกลับโดยไม่ดำเนินการ**

## เลขที่หนังสือ

- รูปแบบ: `ที่ {{รหัสกระทรวง}} {{รหัสกรม}}/{{เลขลำดับ}}`
- ตัวอย่าง: `ที่ ศธ ๐๔๐๐๒/๑๒๓๔` (ศธ=กระทรวงศึกษาธิการ)
- โรงพยาบาล: `ที่ สธ. {{รหัส}}/{{ลำดับ}}` หรือ `ที่ {{รหัส รพ.}} XXX/XXXX`
- ตัวเลข: **ไทย** [๑๒๓]
