# A-Wiki Brain Improvement Gate

Before editing anything that changes how A-Wiki thinks, searches, remembers, delegates, syncs, installs skills/plugins/hooks, or protects private data, read:

`docs/protocols/brain-improvement-gate.md`

## Required Mini-Check

State this before significant edits:

```text
Brain Gate:
- Gain: <clear future-use improvement>
- Shape: <hook/skill/plugin/script/protocol/symlink/action>
- Weight: <why lightweight>
- Safety: <drive/raw/gitignore/public-safe handling>
- Verify: <commands>
```

Then implement the lightest reusable shape that satisfies the request.

## Never Skip

- Cost-first: local/hook/free before expensive primary reasoning.
- Cross-platform: no machine-specific paths.
- Cross-device: preserve sync and `drive/`/`raw/` links.
- Public-safe: no secrets, raw files, private notes, analytics, or personal data in tracked git.
- Package when useful: repeated workflows should become a skill, hook, script, protocol, plugin, or setup step.
