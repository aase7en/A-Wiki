Per ระเบียบสำนักนายกรัฐมนตรีว่าด้วยงานสารบรรณ พ.ศ. 2526. Copy and replace the `{{placeholders}}`.

## ตารางคำขึ้นต้น/คำลงท้ายตามผู้รับ

| ผู้รับ | คำขึ้นต้น (เรียน...) | คำลงท้าย |
|---|---|---|
| พระมหากษัตริย์ / สมเด็จพระบรมราชินี | กราบบังคมทูล (ผ่านราชเลขานุการ) | ควรมิควรแล้วแต่จะทรงพระกรุณาโปรดเกล้าโปรดกระหม่อม ขอเดชะ |
| พระบรมวงศานุวงศ์ชั้นเจ้าฟ้า | ขอพระราชทานกราบบังคมทูล / กราบทูล | ควรมิควรแล้วแต่จะโปรดเกล้าฯ |
| สมเด็จพระสังฆราช | กราบทูล | ควรมิควรแล้วแต่จะโปรด |
| พระสงฆ์ (พระราชาคณะ / ทั่วไป) | นมัสการ | ขอนมัสการด้วยความเคารพอย่างยิ่ง |
| นายกรัฐมนตรี / ประธานองคมนตรี / ประธานรัฐสภา / ประธานวุฒิสภา / ประธานสภาผู้แทนราษฎร / ประธานศาลฎีกา / ประธานศาลรัฐธรรมนูญ / ประธานศาลปกครองสูงสุด / รัฐบุรุษ (และคู่สมรส) | กราบเรียน | ขอแสดงความนับถืออย่างยิ่ง |
| รัฐมนตรี / ปลัดกระทรวง / อธิบดี / ผู้ว่าราชการจังหวัด / นายอำเภอ / ผอ. / หัวหน้า / บุคคลทั่วไป | เรียน | ขอแสดงความนับถือ |

## วลีปิดท้ายมาตรฐาน (Standard closing formulas)

เลือกตามจุดประสงค์ของหนังสือ:

| จุดประสงค์ | วลีปิด |
|---|---|
| แจ้งให้ทราบ | จึงเรียนมาเพื่อโปรดทราบ |
| ขอให้พิจารณา | จึงเรียนมาเพื่อโปรดพิจารณา |
| ขอให้ดำเนินการ | จึงเรียนมาเพื่อโปรดดำเนินการต่อไป |
| ขอความอนุเคราะห์ | จึงเรียนมาเพื่อโปรดพิจารณาให้ความอนุเคราะห์ |
| ขอความเห็นชอบ/อนุมัติ | จึงเรียนมาเพื่อโปรดพิจารณาอนุมัติ |

## หลักการใช้งาน

1. **เลือกประเภทหนังสือ:** ภายนอก (ส่งระหว่างหน่วยงาน) / ภายใน (ในหน่วยงานเดียวกัน — รูปแบบบันทึกข้อความ)
2. **เลือกคำขึ้นต้นตามผู้รับ** จากตารางด้านบน — ผิดระดับ = หนังสืออาจถูกส่งกลับ
3. **หัวเรื่อง:** วลีคำนามสั้น ไม่เกิน 1 บรรทัด ห้ามมีกริยา
4. **เนื้อหา:** ใช้สรรพนาม "ข้าพเจ้า/กระผม/ดิฉัน" ไม่ใช่ "ผม/หนู/เรา"
5. **ห้าม** ครับ/ค่ะ / อิโมจิ / เครื่องหมาย ! / คำทับศัพท์อังกฤษที่มีคำไทยใช้
6. **ปิดด้วยสูตร** "จึงเรียนมาเพื่อโปรด..." เสมอ
7. **ความสม่ำเสมอของตัวเลข:** ใช้เลขไทย ๐-๙ หรือเลขอารบิก 0-9 อย่างใดอย่างหนึ่งตลอดทั้งฉบับ
8. **วันที่:** "วันที่ {{วันที่}} {{เดือนเต็ม}} {{พ.ศ.}}" — เช่น "วันที่ ๒ พฤษภาคม ๒๕๖๙"
