# Embedding Models

> [wiki] Embedding models convert text into vectors for semantic search and retrieval.

## A-Wiki Use

A-Wiki uses multilingual embeddings so Thai and English knowledge can be searched together. Model choice must remain replaceable because embedding support and cost can change.
