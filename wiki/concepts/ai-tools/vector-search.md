# Vector Search

> [wiki] Vector search retrieves semantically similar content, filling gaps where keyword search misses Thai/English wording differences.

## A-Wiki Use

Use vector search as Cost Pyramid Level -1 retrieval alongside FTS5 and graph neighbors. Keep it local-first unless a task explicitly needs external search.
