# การจำแนกประเภทยา (Drug Classification)

## นิยาม

ระบบ ATC (Anatomical Therapeutic Chemical) คือมาตรฐานสากล WHO สำหรับจำแนกยาตามระบบอวัยวะที่ออกฤทธิ์ ใช้เป็น code หมวดใน catalog ของบริษัทขายส่ง

## หมวดยาหลักที่พบในร้านยา (ATC Level 2)

| Code | ชื่อภาษาไทย | ยาที่พบบ่อย |
|------|-------------|-----------|
| **J01** | ยาต้านแบคทีเรีย | Amoxicillin (AMK, AMOXIL, A.M.MOX), Azithromycin (ZITHROMAX), Clarithromycin |
| **N02** | ยาแก้ปวด/ลดไข้ | Paracetamol (ASUMOL, TYLENOL, SARA), Ibuprofen, Mefenamic acid |
| **R05** | ยาแก้ไอแก้หวัด | Ambroxol (AMBROX, AMBRONC), Bromhexine, ACC, Pseudoephedrine |
| **R06** | ยาแก้แพ้ | Chlorpheniramine (CPM), Loratadine, Cetirizine, Fexofenadine |
| **A02** | ยาลดกรด | Omeprazole, Antacid (AMOGIN-GEL, ALUTA), Pantoprazole |
| **C02** | ยาลดความดัน | Amlodipine, Enalapril, Losartan, Valsartan |
| **A11** | วิตามิน/อาหารเสริม | Vitamin C, B complex, Calcium, Folic acid |
| **D07** | ยาสเตียรอยด์ทา | Hydrocortisone, Betamethasone, Clobetasol |
| **D08** | ยาแอนติเซพติค | Povidone iodine, Alcohol, Chlorhexidine |
| **M01** | ยาต้านการอักเสบ | Ibuprofen, Diclofenac, Naproxen |
| **A10** | ยาเบาหวาน | Metformin, Glibenclamide, Insulin |
| **C06/C10** | ยาลดไขมัน | Simvastatin, Atorvastatin, Rosuvastatin |
| **P02** | ยาพยาธิ | Mebendazole (VERMOX), Albendazole |
| **S01** | ยาหยอดตา | Tobramycin, Chloramphenicol, Artificial tears |
| **G03** | ฮอร์โมนเพศ | Norethisterone, Estradiol, Medroxyprogesterone |
