# แนวทางเวชปฏิบัติการให้วัคซีนป้องกันโรคพิษสุนัขบ้า (Rabies PEP/PrEP)

## 1. แนวทางเวชปฏิบัติโรคพิษสุนัขบ้าฯ (ฉบับปรับปรุงครั้งที่ 7)

- **ปีเผยแพร่**: พ.ศ. 2564 (เป็นฉบับมาตรฐานที่กรมควบคุมโรคใช้อ้างอิงปัจจุบัน)
- **หน่วยงาน**: กองโรคติดต่อทั่วไป กรมควบคุมโรค กระทรวงสาธารณสุข
- **URL (verified)**: ✅ <https://ddc.moph.go.th/uploads/publish/1478520231024042510.pdf>
- **Local PDF**: `raw/assets/rabies/ddc-cpg-rabies-2564.pdf` (26 MB)

### สรุปเนื้อหาหลัก
1. **การล้างแผล**: ล้างด้วยน้ำสะอาดและสบู่นานอย่างน้อย 15 นาที + ใส่ยาฆ่าเชื้อ (Povidone Iodine/Alcohol)
2. **สูตร PEP (Post-Exposure Prophylaxis)**:
   - แบบ IM 5 เข็ม: วันที่ 0, 3, 7, 14, 28
   - แบบ ID 4 ครั้ง: วันที่ 0, 3, 7, 28
3. **Rabies Immunoglobulin (RIG)**: ให้เฉพาะ Category III (แผลลึก เลือดออก เลียเยื่อบุ) — ฉีดรอบแผลให้มากที่สุด
4. **การฉีดกระตุ้น**: ถ้าเคยฉีดครบเกิน 6 เดือน → ฉีดกระตุ้น 2 เข็ม (วันที่ 0, 3) ไม่ต้องให้ RIG

## 2. แนวทางการดูแลผู้สัมผัสโรคพิษสุนัขบ้า — สถานเสาวภา (พ.ศ. 2565)

- **ปีเผยแพร่**: พ.ศ. 2565 (สอดคล้องกับ WHO)
- **หน่วยงาน**: สถานเสาวภา สภากาชาดไทย
- **URL (verified)**: ✅ <https://saovabha.org/download/mammal%20bite.pdf>
- **Local PDF**: `raw/assets/rabies/saovabha-rabies-pep-2565.pdf` (1.6 MB)

### สรุปเนื้อหาหลัก
1. **สูตร TRC-ID**: 2-2-2-0-2 (วันที่ 0, 3, 7, 28) ฉีดเข้าในผิวหนังครั้งละ 2 จุด
2. **RIG Infiltration**: ฉีดรอบบาดแผลทุกแผล — **ถ้าเหลือไม่ต้องฉีด IM ส่วนที่เหลือ** (เปลี่ยนจากแนวทางเก่า)
3. **Category II** (ช้ำ/ถลอกไม่มีเลือดออก): ให้เฉพาะวัคซีน ไม่ต้อง RIG
4. **กลุ่มพิเศษ**: ผู้ป่วย HIV ที่ CD4 < 200 → ให้ฉีดแบบ Category III เสมอแม้แผลเล็ก

## 3. WHO Rabies Vaccine Position Paper (April 2018)

- **ปีเผยแพร่**: 2018 (ยังเป็น Position Paper ฉบับล่าสุดที่ WHO ใช้อ้างอิง)
- **หน่วยงาน**: World Health Organization (WHO)
- **URL (verified)**: ✅ <https://cdn.who.int/media/docs/default-source/immunization/position_paper_documents/rabies/pp-rabies-summary-2018.pdf> (summary)
- **Landing**: <https://www.who.int/publications/i/item/who-wer9316>
- **Local PDF**: `raw/assets/rabies/who-rabies-summary-2018.pdf` (665 KB, 3 หน้า — เป็น summary; ฉบับเต็ม WER 9316 อยู่ในหน้า landing)

### สรุปเนื้อหาหลัก
1. **Dose-sparing 1-week ID regimen**: ฉีด ID 2 จุด วันที่ 0, 3, 7
2. **PrEP**: ลดเหลือ 2 ครั้ง (วันที่ 0 และ 7) สำหรับกลุ่มเสี่ยง
3. **RIG**: ฉีดเฉพาะรอบแผลเท่านั้น ไม่ต้องฉีด IM ส่วนที่เหลือ
4. **Previously Vaccinated**: ถ้าถูกกัดซ้ำ → กระตุ้น 1-site ID/IM วันที่ 0 และ 3 ไม่ต้องให้ RIG

## 4. แนวทางเวชปฏิบัติโรคติดเชื้อ พ.ศ. 2567 (Rabies chapter)

- **ปีเผยแพร่**: พ.ศ. 2567
- **หน่วยงาน**: สมาคมโรคติดเชื้อแห่งประเทศไทย × ราชวิทยาลัยอายุรแพทย์
- **URL**: ✅ <https://www.idthai.org/Search/Knowledge/215> (หน้ารวม PDF — ต้องเข้าไปหา PDF เฉพาะบท)

### สรุปเนื้อหาหลัก
1. ยืนยันการใช้สูตรการฉีดวัคซีนตามสถานเสาวภา + WHO
2. เน้นความปลอดภัยของวัคซีนสมัยใหม่ (Cell-culture/Purified) — ใช้ในหญิงตั้งครรภ์/ให้นมได้
3. Human Rabies Case (เริ่มมีอาการแล้ว): เน้น palliative care — Milwaukee Protocol ยังไม่มีหลักฐานชัดเจน
