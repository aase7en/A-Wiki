# Sunday Estate Tax Knowledge Base

> [wiki] ใช้ความรู้ภาษีที่便べ/confirm จาก PDF คู่มือสำหรับบริษัท ซันเดย์เอสเตท

## บทบาท knowledge base นี้

- เป็น lookup reference สำหรับ **business accountant/ops** ของ Sunday Estate
- เน้นที่ **เอกสารที่สามารถลดหย่อนภาษีได้**, **หลักฐานขั้นต่ำ**, **กรณี్యอดจริง**
- อ้างอิง: กรมสรรพากร, **คู่มือการจัดทาเอกสารประกอบการลงบัญชีที่สามารถเป็นรายจ่ายทางภาษีได้** (มีนาคม 2559)

## ข้อมความ/สาระที่ fácil ใช้ได้

### เอกสารหลักฐานขั้นพื้นฐาน (ทุกกรณี)

1. ใบกำกับภาษี / ภาษีมูลค่าเพิ่ม 7% — กรณีผู้ขาย/ให้บริการเป็นผู้ประกอบการ VAT
2. ใบเสร็จรับเงิน — ต้องมีรายการสมบูรณ์ (ชื่อผู้จ่าย / ผู้รับ / จำนวนเงิน / วันที่)
3. ใบสำคัญรับเงิน — ใช้เมื่อผู้รับเงินไม่สามารถออกใบเสร็จรับเงินได้แต่ยอมลงลายเซ็น
4. ใบรับรองแทนใบเสร็จรับเงิน — ใช้เมื่อจ่ายเบ็ดเตล็ดแล้วไม่รุมกลับใบเสร็จ
5. ใบสำคัญจ่าย (Payment Voucher) — กรณีไม่มีหลักฐานอื่น

### ประเภทรายจ่ายพึงบันทึก

| ประเภท | หลักฐานที่ควรมี |
|--------|----------------|
| ค่าซื้อสินค้า/บริการ | ใบกำกับภาษีหรือใบเสร็จ |
| ค่าเช่าอาคาร | สัญญาเช่า + หลักฐานหัก ณ ที่จ่าย |
| เงินเดือนเจ้าของกิจการ | สัญญาจ้างแรงงาน |
| ค่าอาหาร/เดินทาง | มติ/ใบเบิก + ใบสำคัญจ่าย |
| ค่าไฟฟ้า/น้ำ | ใบกำกับชื่อผู้รับจริง + พิสูจน์ว่าบริษัทจ่ายจริง |

## กรณีที่พึงระวัง

- **เอกสาร incomplete** (ไม่ระบุชื่อกิจการที่รับเงิน) → สามารถใช้ใบรับรองแทนใบเสร็จ + Payment Voucher ชดเชยได้
- **ไม่มีหลักฐาน** → ทางภาษีถือ **ไม่ใช่รายจ่าย** ตาม มาตรา 65 ตรี
- ดำเนินการเป็นเท็จ → ระวางโทษตาม ประมวลรัษฎากร / พระราชบัญญัติการบัญชี พ.ศ. 2543
