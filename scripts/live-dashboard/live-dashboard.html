🖥 อ่าน dashboard ยังไง

  • กลางจอ (Network) — การ์ด Primary (🧠 Claude = Senior Critic) เชื่อมเส้นไปการ์ด model ที่กำลังทำงาน; เส้นวิ่ง = กำลัง delegate, หลายเส้นพร้อมกัน = parallel
  • Event Log (ขวา/ล่าง) — timeline เหตุการณ์สด: session start, hook ผ่าน/บล็อก, delegate start/done/fail
  • แถบ hook (บนซ้าย) — hook ที่เพิ่งรัน ✅ ผ่าน / 🔴 บล็อก
  • แท็บ workflow (บน) — 🚀 Session · 📋 Plan · 🧠 Swarm · 💰 Cost-First สว่างเมื่อเข้าโหมดนั้น
  • แถบล่างสุด — แนะนำ model ตามความสามารถ (capability) ต่อชนิดงาน

🎛 เลือก model / ใส่ API key

แท็บ Models: เปิด/ปิด toggle แต่ละ model, แก้ model_id, ดูคะแนน capability (SWE / Term / Repo / reason / speed) แล้วกด บันทึก Models — มีผลรอบ delegate ถัดไป.

แท็บ API Keys: ใส่ key ต่อ provider (โชว์แค่ set/unset ไม่โชว์ค่า). ถ้าไม่ตั้งอะไร = ใช้ default (free model เปิดอยู่แล้ว).

GLM / Z.ai: เปิด toggle GLM 5.2 + ใส่ ZHIPU_API_KEY + ใช้ endpoint api.z.ai (สากล) แก้ model_id เป็น glm-5.2 ได้.

💰 Cost-first (สำคัญ)

การจัดลำดับยึด cost-first เสมอ — model เสียเงิน ไม่แซง model ฟรี. capability สลับลำดับเฉพาะ ภายใน cost class เดียวกัน.
