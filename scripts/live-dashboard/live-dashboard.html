🖥 อ่าน dashboard ยังไง

  • 🌊 Flow — จุดกำเนิด (🧠 Senior Critic) ส่งข้อมูลไหลทางท่อไปยังแต่ละ model; ท่อสว่าง + อนุภาค = กำลังทำงาน

  • 🔗 Graph — knowledge-graph ของ session/task/agent
  • 🧩 Skills — catalog ของ skill ทั้งหมด พร้อมคำอธิบายไทย + ตัวอย่าง + simulation; filter ตาม agent เพื่อดูความสามารถของ agent ที่คุณใช้
  • Event Log — timeline เหตุการณ์สด · แถบ hook ✅/🔴 · cost tier — L-1 ถูก → L4 แพง
