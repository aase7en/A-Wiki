# /build — Implement slice by slice

Maps to: `skills/engineering-lifecycle/build/incremental-implementation/SKILL.md` + `skills/engineering-lifecycle/build/test-driven-development/SKILL.md`

## When to use
- You have approved tasks and are ready to write code
- Any change touching more than one file

## Flow
1. Run `incremental-implementation` skill (thin vertical slices)
2. For each slice: implement, test, verify, commit
3. Run `test-driven-development` skill alongside (Red-Green-Refactor)
4. Each increment leaves codebase in a working state
