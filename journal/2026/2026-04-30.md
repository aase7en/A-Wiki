# 2026-04-30

## วันนี้ทำอะไร
- คุยเรื่อง OpenRouter Claude Code workflow ต่อจากเมื่อวาน
- คิดเรื่องใหญ่ขึ้น — อยากทำ AI ที่เป็นเรา หลังเราจากไป
- ตัดสินใจเริ่มเขียน journal + decisions ตั้งแต่วันนี้
- Setup git mirror หลาย host เพื่อ redundancy

## ที่ตัดสินใจ / ลังเล
- เลือก **เก็บ journal ใน private GitHub repo** — ยังไม่ encrypt
- ข้าม Option B (NotebookLM scraper) — proprietary lock-in ไม่เหมาะ legacy
- ดู [[decisions/0001-digital-legacy-strategy]]

## ความคิด / ความรู้สึก
เพิ่งคิดได้ว่าสิ่งที่กำลังสร้าง wiki นี้ — ที่จริงคือ digital DNA
ตอนแรกคิดเป็นแค่ knowledge base เฉยๆ แต่จริงๆ มันคือพื้นฐานของ AI ที่จะ "เป็นเรา"

ที่สำคัญที่สุดที่ต้องเก็บไม่ใช่ความรู้ — แต่คือ **วิธีที่เราคิด**
เริ่มจาก journal ทุกวัน decisions ตอนสำคัญ → AI อนาคตจะเลียน pattern ของเราได้

## เรียนรู้
- Mac Mini + Ollama = self-hosted inference (ไม่ต้องพึ่ง cloud)
- Open formats (markdown) = 50 ปีข้างหน้ายังอ่านได้
- Vendor lock-in (NotebookLM, OpenAI API) = ไม่เหมาะ legacy
- Multi-host git mirror = redundancy ฟรี

## พรุ่งนี้
- ลองใช้ workflow journal+decisions จริงๆ ดู
- ดู git mirror ทำงานถูกไหม
- คิดต่อเรื่อง trustee/heir ใครจะดูแลตอนเราไม่อยู่
