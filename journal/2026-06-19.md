## Context
Hermes Docker on Umbrel Pi 5 serves Chat API (OpenAI-compatible) on port 8501, but Docker networking
doesn't forward to external interfaces. `/health` + `/v1/chat/completions` work on localhost and same-LAN,
but Tailscale IP gets empty reply. Tailscale Funnel works as workaround.
awiki-live.html hardcoded one URL with no fallback — needed profile system + auto-detect.

## Root Cause
Umbrel Docker containers isolate ports by default. The `docker-compose.yml` for hermes-agent needs
explicit `ports: ["8501:8501"]` mapping. Without it, only Docker-internal networking sees the port.
